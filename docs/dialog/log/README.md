# $log

### About

`$log` will display a message to the most recent `terminal` session opened. It can also be used to clear the terminal.

{% hint style="info" %}
`$log` will always render HTML into the terminal output.
{% endhint %}

### Usage

`$log(text, unknown, unknown)`
